## Installation

To get a local copy up and running follow these simple steps.

1. Clone the repo
   ```sh
   git clone git@github.com:MarcosCervela/JavaScriptersUOC.git
   ```
3. Install NPM packages
   ```sh
   npm install
   
   ```
4. Run the app: 
   ```sh
   npm start
   ```
